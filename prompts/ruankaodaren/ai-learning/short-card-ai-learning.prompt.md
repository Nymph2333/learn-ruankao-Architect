# Short Card AI Learning Prompt

## Boundary

You may only use the Source Layer and Source Packet as source material. AI output must be explicitly labeled `AI-generated`. Do not modify the Source Layer, do not write AI content into `source_content`, and do not treat generated explanation as original source text.

Forbidden actions:

- Do not OCR.
- Do not reconstruct image tables.
- Do not decrypt encrypted XHR.
- Do not directly read raw HTML or raw XHR.
- Do not access webpages.
- Do not claim image content has been identified unless a human reviewer provided that interpretation.

## Short-Source Rules

- The source is verified short text.
- AI may expand concepts, but every expansion must be marked `AI-generated`.
- Do not claim the source covered all topic details.
- If taxonomy is suspect or multi-card sequence is possible, do not claim completeness and require manual review.

## Output Structure

### Source Summary / 原文摘要

Summarize only the short source text and source packet metadata.

### AI Explanation / AI解析

Provide AI-generated concept expansion without claiming source completeness.

### Architecture Mapping / 架构师考点映射

Map the short source to architecture exam points as AI-generated guidance.

### Case Study Pattern / 案例答题模式

Draft case-study answer patterns as AI-generated material.

### Paper Usage / 论文表达

Draft paper expressions as AI-generated material.

### Misconceptions / 易错点

List likely mistakes, marking them as AI-generated.

### Memory Hooks / 记忆钩子

Create memory aids as AI-generated learning support.

### Review Notes / 复核提示

Record short-source and taxonomy review warnings.
